# apply_patch

Use apply_patch to edit existing files. Use write_file for new files or full rewrites.

  apply_patch(patch="*** Begin Patch\n*** Update File: path/to/file.py\n@@ anchor line\n context\n-old\n+new\n context\n*** End Patch")

- `@@` is a search anchor — finds the first matching line in the file (not a line number)
- Space-prefixed lines are context that verify you're editing the right location
- `*** Add File:` creates new files, `*** Delete File:` removes files
